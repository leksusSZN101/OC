3.  notepad++

Notepad++ — это бесплатный и открытый редактор исходного кода, написанный на C++ и работающий на платформах Microsoft Windows. Он значительно превосходит стандартный Блокнот Windows по функциональности и удобству использования, особенно для программистов и веб-разработчиков.

Работает только на Windows: Notepad++ не доступен для других операционных систем (macOS, Linux). Для них существуют аналоги с похожим функционалом.

В итоге, Notepad++ является отличным выбором для редактирования кода, особенно для начинающих программистов, а также для тех, кто нуждается в простом, но эффективном редакторе текста с поддержкой подсветки синтаксиса.
