5.  Brackets

Brackets — это бесплатный, открытый исходный код текстовый редактор, ориентированный прежде всего на веб-разработку. Его сильная сторона — удобство работы с HTML, CSS и JavaScript, хотя он поддерживает и другие языки программирования с помощью плагинов.

  • 1.   Живой просмотр
  • 2.  Интегрированный редактор препроцессоров
  • 3.  Поддержка расширений
  • 4.  Интеграция с Git
  • 5.  Простота использования

Brackets — хороший вариант для веб-разработчиков, которым нужен лёгкий, быстрый и интуитивно понятный редактор с удобным живым просмотром

Он идеально подходит для небольших проектов или для тех, кто предпочитает минималистичный интерфейс.
